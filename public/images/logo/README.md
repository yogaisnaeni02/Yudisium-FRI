# Logo Files

Silakan letakkan file logo berikut di folder ini:

1. **fri-logo.png** - Logo lengkap Fakultas Rekayasa Industri (untuk header/sidebar)
   - Ukuran disarankan: 200x200px atau lebih besar (akan di-resize otomatis)
   - Format: PNG dengan transparansi (background transparent)
   
2. **fri-icon.png** - Icon/logo kecil (untuk favicon dan icon kecil)
   - Ukuran disarankan: 64x64px atau 128x128px
   - Format: PNG dengan transparansi

3. **favicon.ico** - Favicon untuk browser
   - Ukuran: 16x16px, 32x32px, atau 48x48px
   - Format: ICO

## Cara Upload Logo

1. Simpan logo Fakultas Rekayasa Industri sebagai `fri-logo.png` di folder `public/images/logo/`
2. Simpan icon sebagai `fri-icon.png` di folder `public/images/logo/`
3. Simpan favicon sebagai `favicon.ico` di folder `public/` (ganti yang lama)

Jika file tidak ditemukan, sistem akan menggunakan fallback SVG logo.
